# Screen 05: Victory

Shown when the player is the last survivor.

## Purpose

- Celebrate the win (Gold/Dark theme).
- Display match statistics.
- **Drive retention** by encouraging the winner to Host the next game.

## Key Elements

| Element | Description |
|---------|-------------|
| **Victory Card** | Crown icon, "Victor" title, stats grid. |
| **Narrative** | Final epilogue text. |
| **Action Buttons** | Host Next Game, Play Again. |

## Navigation

- **Host**: Goes to `015-host-lobby`.
- **Play Again**: Goes to `01-tribute-setup` (Solo).

## Viewing

```bash
pnpm dev:pwa
# Open: http://localhost:5173/uiux/05-victory/index.html
```
